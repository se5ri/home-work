# 수업 회고

이번 주 수업은 강의 속도가 빨라서 이해하면서 배우기보단 일단 배우고 회고 시간 이용해서 공부했었다.
그래서 이번 주 내용을 회고할 때 블로그를 써보았고 원래는 코드 위주로 정리했다면 이번 주는 개념 위주로 정리하고 코드를 보면서 실습해 보는 식으로 공부했다.
회고 시간에 팀원들이랑 ui 실습을 한 게 많이 도움이 되었고, 수업 후 스터디가 도움이 많이 되었다
